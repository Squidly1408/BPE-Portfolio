Help & Guidance

Support for Big Picture students building official portfolio sites.

Quick Start

1. Create an account → 2. Create portfolio in Dashboard → 3. Open Editor → 4. Customize flower/theme/content.

Publishing Guidance

Use one official portfolio with updated projects, internships, and reflection evidence each term.

Recommended Workflow

Set defaults in Settings, build in Dashboard, then refine final design/content in the Editor.

FAQ

Can I create multiple portfolios?

Yes, one for each year, pathway, or presentation.

Can I change themes later?

Yes, themes can be switched in editor controls at any time.

How do I edit the BPE flower?

Open the editor and use the flower petal sliders in the Profile section.
